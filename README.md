# 🔐 Login System - Python

A simple and secure login system built with Python where the user must authenticate using a password.

## 🚀 Features

- Hidden password input (getpass)
- Maximum attempt limit (10 tries)
- Input validation (prevents empty input)
- Secure login logic
- User-friendly messages

## 🛠️ Technologies Used

- Python
- getpass module

## ▶️ How to Run

1. Clone the repository
git clone https://github.com/joaoleaodev/login-system-python.git

2. Navigate to the folder
cd login-system-python

3. Run the program
python login_system.py

## 🎮 Game Rules
